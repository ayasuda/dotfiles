# Atsushi Yasuda's dotfiles

いろんなシステムで使う用の設定ファイル集
とりあえず、 `Debian ` で動く。

## Install

```terminal
git clone git://github.com/yasuda-atsushi/dotfiles ~/.dotfiles
cd ~/.dotfiles
sh ./install.sh
```
